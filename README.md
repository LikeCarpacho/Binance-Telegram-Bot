## Commands
1. /see : consult the binance leaderboard
2. /follow : follow binance trader
3. /following : consult your following list
4. /startbot : start the tracking bot
5. /stopbot : stop the tracking bot

## Starting
1. run: `npm i`
2. edit: `config.js`
3. run `npm start`
